## Installation
Follow these steps to install and set up the project:

```bash
git clone https://github.com/viktor-dimalanta/full-stack-engineering-test.git
cd your-repo
```

## Usage

Start the front-end

```
cd front-end
npm start
```

Start the back-end

```
node server.js
```
